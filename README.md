# Smart-Study-Planner-Java

## Project Overview

Smart Study Planner is a console-based Java application designed to help students organize subjects, add topics, track completion, and monitor study progress.

This project demonstrates object-oriented programming concepts and file handling in Java.


## Features

- Add new subjects
- Add topics under subjects
- Mark topics as completed
- View progress percentage
- Save study data to file (data.txt)



## Concepts Used

- Classes and Objects
- ArrayList
- OOP (Encapsulation)
- File Handling (FileWriter)
- Loops and Conditional Statements
- User Input using Scanner



## How to Run

1. Compile the program:

   javac SmartStudyPlanner.java

2. Run the program:

   java SmartStudyPlanner



## Output File

The program stores data in:

data.txt
